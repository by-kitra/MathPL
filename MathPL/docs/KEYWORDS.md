# 🔑 Ключевые слова MathPL

> ⚠️ **Ранний доступ**: Некоторые ключевые слова ещё не реализованы.  
> См. [ROADMAP.md](ROADMAP.md) для статуса функций.

---

## Структура программы

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `task` | ✅ | Начало именованной задачи | `task main:` |
| `task#` | ✅ | Начало безымянной задачи | `task#` |
| `close` | ✅ | Конец задачи | `close` |
| `base` | ✅ | Объявление переменных | `base = a, b, c` |
| `and` | ✅ | Разделитель в base и тензорах | `1 and 2 and 3` |

---

## Типы данных и значения

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `yes` | ✅ | Булево Истина | `flag = yes` |
| `no` | ✅ | Булево Ложь | `flag = no` |
| `nil` | ✅ | Отсутствие значения | `value = nil` |
| `true` | ⬜ | Алиас для yes (в планах) | `flag = true` |
| `false` | ⬜ | Алиас для no (в планах) | `flag = false` |

---

## Операторы и выражения

| Ключевое слово/Символ | Статус | Описание | Пример |
|------------------------|--------|----------|--------|
| `+` | ✅ | Сложение / конкатенация | `a + b` |
| `-` | ✅ | Вычитание | `a - b` |
| `*` | ✅ | Умножение (поэлементное) | `a * 2` |
| `/` | ✅ | Деление | `a / 2` |
| `^` | ✅ | Возведение в степень | `a ^ 2` |
| `@` | ✅ | Матричное умножение | `A @ v` |
| `\|>` | ✅ | Конвейер данных | `data \|> fn` |
| `==` | ✅ | Проверка равенства | `a == b` |
| `!=` | ✅ | Проверка неравенства | `a != b` |
| `>` | ✅ | Больше | `a > b` |
| `<` | ✅ | Меньше | `a < b` |
| `>=` | ✅ | Больше или равно | `a >= b` |
| `<=` | ✅ | Меньше или равно | `a <= b` |
| `=` | ✅ | Присваивание | `x = 42` |

## Управление потоком

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `if` | ✅ | Условие | `if x > 10:` |
| `else` | ✅ | Иначе | `else:` или `else if:` |
| `for` | ✅ | Цикл | `for item in list:` |
| `in` | ✅ | Оператор вхождения (в цикле) | `for x in items` |
| `while` | 🟡 | Цикл while (частично) | `while условие:` |
| `->` | ✅ | Возврат из функции | `-> результат` |

## Функции

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `fn` | ✅ | Объявление функции | `fn add(a, b) -> a + b` |
| `funct` | ⬜ | Модуль с состоянием | `funct counter:` |
| `function` | 🟡 | Указатель на функцию/модуль | `if work function =` |

## Вывод

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `say` | ✅ | Вывод значения | `c = a + b = (say "Сумма")` |
| `вывод` | ✅ | Русский алиас say | `вывод "Привет"` |

## Координаты и движение

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `coor` | ✅ | Система 2D координат | `coor to (x;y)` |
| `coor3d` | 🟡 | Система 3D координат | `coor3d to (x:y:z)` |
| `to` | ✅ | Абсолютное перемещение | `coor to (10;20)` |
| `by` | ✅ | Относительное перемещение | `coor by (1;0)` |
| `launch` | 🟡 | Инициализация пространства | `launch centre = 3D` |
| `centre` | 🟡 | Центр координат | `launch centre = 3D` |
| `forward` | 🟡 | Направление вперёд | `function to forward` |
| `left` | 🟡 | Направление влево | `function to left` |
| `back` | 🟡 | Направление назад | `function to back` |
| `right` | 🟡 | Направление вправо | `function to right` |
| `move` | 🟡 | Движение | `move LiB` |
| `moveLiB` | 🟡 | Библиотека движения | `function and moveLiB` |

## Ввод и клавиши

| Ключевое слово | Статус | Описание | Пример |
|----------------|--------|----------|--------|
| `request` | 🟡 | Запрос | `request access keyboard` |
